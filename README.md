# Vivero Lazulada

Sitio web del Vivero Lazulada, especializado en cerco vivo con más de 25 años de experiencia.

🌐 **Web:** [https://ivo-bisc.github.io/ViveroLazulada/](https://ivo-bisc.github.io/ViveroLazulada/)

## Desarrollo local

```bash
npm install
npm run dev
```

## Deploy a GitHub Pages

Para publicar cambios en la web:

```bash
git add . && git commit -m "descripción del cambio" && git push origin main && npm run deploy
```
